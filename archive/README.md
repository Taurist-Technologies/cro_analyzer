# Archived Files

This directory contains legacy and rarely-used scripts that have been archived during the codebase reorganization.

## Contents

- **Architecture.py** - Documentation diagram generator (ASCII architecture visualization)
- **count_issues.py** - One-off analytics script for counting CRO issues
- **download_audits.py** - Utility script for downloading audit files

## Note

These files are kept for reference but are not part of the active codebase.
If you need functionality from these files, consider:
1. Moving them back to an appropriate location
2. Extracting specific functions into the main codebase
3. Updating imports if dependencies have changed
