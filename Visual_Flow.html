Features

  • Automated website screenshot capture
  • AI-powered CRO analysis using Claude's vision
  • 2-3 actionable issues per analysis
  • Specific recommendations for each issue
  • Full-page screenshots included in response
  • JSON format ready for n8n integration
  • RESTful API with FastAPI
  • Async processing for better performance
  • Health check endpoint for monitoring
  • Docker support for easy deployment

Getting Started

Follow these simple steps:

  1. Install dependencies: pip install -r requirements.txt
  2. Install Playwright: playwright install chromium
  3. Set API key: export ANTHROPIC_API_KEY='your-key'
  4. Start service: python main.py
  5. Test it: python test_service.py
  6. Integrate with n8n using the example workflow
